# 🎉 Big Jumps 4 TODs Website

**Business Name:** Big Jumps 4 TODs
**Industry:** Soft Play / Baby Gym Rentals
**Location:** New Haven, CT

This website was built for a **local client** to provide a **safe and fun environment for infants and toddlers to play, explore, and enjoy**. The platform simplifies the booking process for soft play and baby gym rentals while helping the business manage schedules efficiently.

---

## 🛠 Features

### Customer-Facing Features
- 🎈 Browse and view available soft play and baby gym rental options
- 📅 Book schedules for bouncy houses and other baby gym rentals for events
- ✉️ Submit booking requests easily through a user-friendly interface
- ⏱ See availability in real-time to avoid double bookings

### Admin Features
- 🗂 View and manage all rental requests
- ✅ Accept or ❌ deny requests based on availability
- 📊 Keep schedules organized and prevent conflicts

---

## 💻 Technologies Used
- **Next.js** – React framework for server-side rendering and fast performance
- **React** – Interactive and dynamic UI components
- **Express.js** – Backend server handling API requests and business logic
- **PostgreSQL** – Database for storing bookings and availability
- **CSS / Tailwind CSS** – Styling and responsive design
- **JavaScript** – Core programming language

---

## 🎯 Purpose
The website helps **Big Jumps 4 TODs** streamline its booking process. Customers can book soft play and baby gym rentals effortlessly, while the admin panel allows the business to manage bookings efficiently and safely.

---

## 🚀 How to Use
1. Visit the website and browse available rental packages.
2. Select a date and time for your event.
3. Submit a booking request.
4. Admin reviews and accepts/denies the request based on availability.
5. Receive confirmation and enjoy a safe, fun play experience for children.
